How It Works

The chatbot follows a simple support flow to understand the customer query and reply with helpful guidance.

1

Ask a Query

The user enters a question related to refund, order problem, complaint, or payment issue.

2

Intent Detection

The chatbot analyzes the message using Dialogflow ES and matches it with the correct support intent.

3

Bot Response

It sends a relevant response based on the user's issue and provides the next step or solution.

4

Quick Assistance

The user gets fast support without needing to wait for manual help in basic service situations.
